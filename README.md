# Naturaleza-muerta
En la pintura, donde la naturaleza muerta tiene una larga tradición, se le da este nombre a un cuadro con objetos inanimados. Lo mismo sucede cuando la fotografía tiene un sentido más amplio y no se refiere únicamente a uno sino también a la representación de la naturaleza muerta de tipo artístico, sensación fotográfica de objetos con fines publicitarios, editoriales, etc.
